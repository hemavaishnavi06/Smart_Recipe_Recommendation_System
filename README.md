# Smart Recipe Recommendation System

A C++ console-based application that recommends recipes based on ingredients, category, diet type, cooking time, and ratings.

## Features

- User Registration & Login
- Recipe Management
- Recipe Recommendation
- Meal Planner
- Favourite Recipes
- Meal History
- CSV File Storage

## Technologies

- C++
- Object-Oriented Programming
- STL (Vector, Map)
- File Handling
- CSV Database
